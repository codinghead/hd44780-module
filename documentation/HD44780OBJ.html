HD44780OBJ Structure

New data type HD44780OBJ

Syntax
C++
typedef struct HD44780OBJTYPE {
  HLCDIF hLcdIf;
  LCDIFFP * lcdIfFunctionPointers;
  HD44780NUM hd44780Num;
  unsigned char hd44780Flags;
  struct HD44780OBJTYPE * nextHD44780Obj;
} HD44780OBJ;
File

HD44780.h

Description

This is the main object for storing the context of the HD44780 LCD chipset which is currently in use. Members are:

  • hLcdIf - Handle to the LCD interface to which this HD44780 controller is connected
  • * lcdIfFunctionPointers - Pointer to the functions that implement the low level access to the LCD data bus
  • hd44780Num - The number assigned to this HD44780 controller
  • hd44780Flags - Flags used by the module to keep track of the status of this HD44780 controller (private to this module)
  • *nextHD44780Obj - Pointer to the next HD44780 object in the linked list
